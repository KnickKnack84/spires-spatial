# Spires Spatial — website

Marketing website for **Spires Spatial** (Hal Spires) — 3D property tours, dollhouse views, measured floor plans, and digital twins for real-estate listings in Middle Georgia.

## What this is
A single self-contained static page (`index.html`) — all CSS and JS inline, all imagery/fonts CDN-hosted. No build step, no dependencies. Opening `index.html` in any browser runs the full site.

## Live preview
Deployed via GitHub Pages — see the repo's Pages settings for the URL.

## Stack & conventions
- Static HTML/CSS/JS, single file.
- Type: Fraunces (display) · Instrument Sans (body) · JetBrains Mono (labels).
- Palette: ink `#14110F` · warm bone `#F4EFE6` · bronze `#A87B4F` · aged-gold `#C9A961`.
- Hero is a 3-mode interactive viewer (Photo · Dollhouse · Floor Plan) mirroring the Matterport/Asteroom product UI; uses the View Transitions API for cross-fades.
- "How it works" steps use hand-coded SVG line illustrations (no photos).
